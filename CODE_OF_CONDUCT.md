# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

In short:

- Be welcoming, inclusive, and respectful.
- Disagreement on technical merit is encouraged; disagreement on identity is not.
- Harassment, discriminatory language, doxxing, and personal attacks are prohibited.
- Maintainers will moderate as needed.

## Scope

This code of conduct applies to all project spaces — issues, PRs, commits, discussions, and any social-media or chat presence operated under the project's name.

## Reporting

If you experience or witness unacceptable behavior, please email the maintainer privately. Reports will be reviewed promptly and confidentially.

## Enforcement

Maintainers will follow the Contributor Covenant v2.1 enforcement guidelines. Consequences range from a private warning to a permanent ban depending on severity and context.
